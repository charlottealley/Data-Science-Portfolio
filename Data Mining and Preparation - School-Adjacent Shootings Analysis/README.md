# Data Mining and Preparation – School-Adjacent Shooting Analysis
**Handling Sensitive Data, Ethically**

This project demonstrates data acquisition and preparation using three distinct data sources: a flat file, an API, and web scraping. Four notebooks document the complete workflow, from data collection and cleaning to dataset integration and storage within a SQLite database. Special attention is given to ethical considerations when working with sensitive violence-related data.

## Key Techniques
- Data mining from flat files, APIs, and web sources
- Data cleaning and standardization
- Dataset integration and validation
- SQLite database design and implementation
- Ethical handling of sensitive data

## Data Science Impact
High-quality analysis begins with high-quality data. This project highlights the foundational skills required to transform disparate real-world datasets into a reliable analytical resource.
