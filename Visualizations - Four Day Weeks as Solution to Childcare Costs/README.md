# Visualizations – Four Day Weeks as a Solution to Childcare Costs
*Using visual storytelling to communicate complex social challenges.*

This project combines exploratory data analysis with visualization design to examine four-day work weeks as a potential response to rising childcare costs. Two notebooks support the analysis: one focused on data exploration and another dedicated to creating communication assets for a broader public-facing campaign.

## Key Techniques
- Exploratory data analysis (EDA)
- Data visualization design
- Dashboard development
- Infographic creation
- Short-form video storytelling
- Data communication

## Social Impact
Effective visualizations can transform complex datasets into accessible insights. This project uses data storytelling to encourage discussion around workplace flexibility and childcare affordability.
