# Time to Event – Predicting All-Cause Hospital Readmissions with NLP and Cox Modeling
**Using clinical narratives to improve readmission risk prediction.**

This project combines structured electronic health record data with natural language processing techniques to predict 30-day hospital readmissions. Sentence embeddings derived from clinical narratives are incorporated into a Cox survival model to estimate readmission risk and identify patients who may benefit from targeted interventions.

## Key Techniques
- Natural language processing (NLP)
- Sentence embeddings
- Principal Component Analysis (PCA)
- Cox survival modeling
- Time-to-event prediction
- Risk stratification

## Healthcare Impact
Reducing preventable hospital readmissions improves patient outcomes while lowering healthcare costs. This project demonstrates how structured and unstructured clinical data can be leveraged together to support proactive care decisions.
