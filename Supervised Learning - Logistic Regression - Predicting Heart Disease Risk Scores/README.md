# Heart Disease Risk Prediction
**Handling Sensitive Data, Ethically**

Using RandomizedSearchCV to compare and select from four distinct classification models and their hyperparameters, this machine learning project optimizes heart disease risk scores to clinically diagnose patients in a screening environment. The final logistic regression model uses a single test feature and two binary features (yes/no questions) to predict risk. 

## Key Techniques
- Model comparison: Logistic Regression, SVM, Random Forest, XGBoost
- Hyperparameter optimization via RandomizedSearchCV
- Feature importance analysis 
- Clinically practical: **3 total features** for more cost and time-efficient screenings

## Healthcare Impact
Streamlined cardiac risk communication enables much-needed preventive care.
