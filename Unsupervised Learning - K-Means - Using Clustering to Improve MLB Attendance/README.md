# Unsupervised Learning – K-Means – Using Clustering to Improve MLB Attendance
*Using data-driven segmentation to better understand fan behavior.*

This project applies unsupervised learning techniques to Major League Baseball attendance data. Through exploratory data analysis, descriptive statistics, and K-Means clustering, the analysis identifies meaningful attendance patterns and generates recommendations for improving fan engagement and attendance outcomes.

## Key Techniques
- Exploratory data analysis (EDA)
- Descriptive statistics
- K-Means clustering
- Cluster interpretation and profiling
- Business-focused recommendations

## Business Impact
By identifying distinct attendance patterns, organizations can tailor promotions, scheduling decisions, and marketing efforts to better serve different fan segments.
