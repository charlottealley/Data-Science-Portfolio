# Big Data – SQL, Spark, and Hadoop
*Building an end-to-end analytics pipeline across the Hadoop ecosystem.*

This project simulates a big data workflow for predicting student GPA using sleep-related features. Data is ingested through Apache NiFi, stored in HDFS, queried through Hive, analyzed using Spark SQL and MLlib, and ultimately written to HBase. YARN manages cluster resources throughout the pipeline.

## Key Techniques
- Apache NiFi data ingestion
- HDFS distributed storage
- Hive data warehousing
- Spark SQL and MLlib
- HBase NoSQL storage
- YARN resource management

## Big Data Impact
The project demonstrates how multiple technologies can work together to support scalable analytics, machine learning, and operational reporting within a distributed environment.
