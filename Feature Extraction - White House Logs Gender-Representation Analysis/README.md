# Feature Extraction – White House Logs Gender-Representation Analysis
*Transforming messy text into meaningful insights.*

This project explores representation within White House visitor logs through feature extraction and text analysis. Following extensive exploratory data analysis, a gender feature is derived from unstructured data and used to examine representation patterns across commonly and less commonly used White House rooms.

## Key Techniques
- Exploratory data analysis of messy text data
- Feature engineering and extraction
- Data quality assessment
- Representation analysis
- Communication through data storytelling and infographic design

## Social Impact
Understanding representation patterns can help organizations identify potential disparities and encourage more inclusive decision-making.
